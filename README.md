# A N D R O I D  |  B U I L D E R (Work in progress)
## Inspired by @akhilnarang

How to use this script?

```
git clone https://github.com/Nandanrmenon/scripts.git
```

Then run 
```
cd script 
. setup.sh
```

Currently Supported ROMs :
* AICP
* AIM ROM
* Ancient OS
* AOSIP
* AOSP Extended
* Arror OS
* Beast ROM
* Bliss ROMs
* Bootleggers
* Colt OS
* Corvus OS
